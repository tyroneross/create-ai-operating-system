# raw/personal/

Use this folder only in a private workspace.

## What Could Go Here

- Goal notes.
- Reflection notes.
- Personal operating observations.
- Private planning material.

## Privacy Gate

Do not publish journals, addresses, phone numbers, family details, identity documents, health details, financial records, screenshots, or private correspondence.

## What To Write In The Wiki

- Lessons.
- Decisions.
- Review cadence.
- Reusable patterns.
- Open questions.
