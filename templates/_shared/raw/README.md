# raw/

Raw files are immutable source material. The wiki is the distilled operating layer.

## Flow

1. Drop new files into `raw/ingest/`.
2. Summarize them into `wiki/sources/`.
3. Link durable ideas into `wiki/concepts/`.
4. Rebuild the index.
5. Keep raw files available for verification.

## Public Safety

Do not commit sensitive raw files to a public repository. Use placeholders in this template and keep real content in a private clone.
