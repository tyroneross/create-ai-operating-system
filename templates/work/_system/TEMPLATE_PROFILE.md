# Work Template Profile

## Use This Template For

- Project operating systems.
- Strategy and research workspaces.
- Meeting memory and decision continuity.
- Customer, stakeholder, or initiative context.
- Team knowledge bases that need a clear source lifecycle.

## Default Work Page Types

- Project page.
- Decision log.
- Meeting synthesis.
- Operating review.
- Research brief.
- Reusable template.

## Guardrails

- Do not commit confidential company, customer, or employee data to a public repo.
- Sanitize examples before sharing.
- Use generic roles like `Example Sponsor`, `Example Customer`, and `Example Project` in public artifacts.
- Label assumptions and confidence in decision pages.
