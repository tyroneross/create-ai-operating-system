# raw/meetings/

Use this folder for meeting source material in a private workspace.

## What Could Go Here

- Agendas.
- Notes.
- Transcripts.
- Follow-up lists.

## What To Write In The Wiki

- Bottom line.
- Decisions.
- Action items.
- Risks.
- Open questions.
- Pages that should be updated.

## Tradeoff

Transcript-level detail is useful for audit, but meeting synthesis should be short enough to drive execution.
